How search works

If you try to search with empty search field, system return you random images. You'll get new portion of random images on every page refresh.

You can enter one or more tags into search field and system will try to find results correspondingly.

Search engine is strict, it expects you to enter exact tags, no spelling mistakes, no similar words. Exact tag or nothing. You can browse all existing tags on the tags page.

You can also specify your query by using + as and, - as not and | as or. For example fat cats | small dogs + frogs - snakes will be treated as anything with tag "fat cats" or with tag "small dogs", obligatory with tag "frogs" and definitely without tag "snakes".

If you're getting strange results and want to ensure that system understood your request, you can click on note under search field (works only if you're actually searched something). This will toggle explanation section with parsed request and timing for each operation. This allows you to verity that your request is parsed correctly.

First phrase of a query implicitly gets its keyword. You can explicitly specify it by entering something like "+ cats + dogs" or "- cats + dogs".

or fat cats or small dogs and frogs not snakes

Each and is treated as "result must include this tag". More tags you enter, more specific the result will be.

Each not is treated as "result must not include this tag". More tags you enter, less results you get.

When working with or, search engine combines all records, that have at least one of the given tags. You will see difference only entering more than one or statement, otherwise result will be the same as entering and.

Tags page gives you easy access to all available tags, ordered by alphabet.

Number after tag box show how many records are using this tag (numbers in this example are arbitrary).

How navigation page works

Navigation page allows you to specify which themes you are interested in.

By clicking on checkboxes you can exclude some themes from search. Excluded theme will appear crossed out and will not be used during search.

Additionally this will affect:

  • No materials from theme will appear during random search (when there is no user query) and standard search.
  • Items from excluded theme will not be counted in statistics on the tags page.
  • Tags from excluded theme will not be displayed on the tags page.

By clicking on the arrow in the right corner of every theme you can get list of all groups in that theme.

Beware: you can get strange results clicking on groups in excluded themes. Usually you will find nothing. Include theme before browsing its groups.

How feedback page works

Feedback page was added so users could mention which material they'd like to see on site and also note bugs that should be fixed. You can submit any thoughts related to the site, what you liked, what not and which materials you see as not appropriate. Don't forget to mention your contact address if you want to get response. Usually inbox gets checked once a week, so do not expect a fast reply.
